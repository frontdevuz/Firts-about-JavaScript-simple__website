Qiymatlar va O'zgaruvchilar

Qiymat bu eng kichik ko'rinishdagi ma'lumot yoki ma'lumot bo'lagi!

O'zgaruvchi (variable) bu o'zida ma'lum bir qiymatni saqlovchi konteyner (container) ga aytiladi!

O'zgaruchilarni e'lon qilingandan so'ngina ishlatish imkoniyati mavjud .E'lon qilinmagan o'zgaruvchilarni ishlatish xatolikka olib keladi

Data types (Ma'lumot turlari) in Java script

Java script da qiymatlar ikki turga bo'linadi bular : primative va object

Primative turdagi qiymat : To'g'ridan-to'g'ri o'zgaruvchi murojat qiladigan manzilda joylashadi:

Bularga misol qilib biz :

  • Number: sonlarni o'zida saqlovchi ma'lumot turi
  • String: o'zida belgilar ketma - ketligini saqlovchi ma'lumot turi
  • Boolean: mantiqiy ma'lumot turi bo'lib faqatgina true (rost) yoki false (yolg'on) qiymatini qabul qilish mumkin bo'lgan ma'lumot turi
  • Undefined: ha'li aniqlanmagan ma'lumot turi
  • Null: bo'sh qiymatga ega bo'lgan ma'lumot turi
  • Symbol: yagona (unique) va o'zgarmas qiymatga ega bo'luvchi ma'lumot turi
  • BigInt: katta sonlarni saqlashga mo'ljallangan ma'lumot turi

    • Primative turdagi ma'lumot turlari - 7xil bo'ladi!

    Javascript dynamic typing funksiyasiga ega bo'lib uning yordamida o'zgaruvchi saqlovchi ma'lumot turini ko'rsatmasdan ham qiymat saqlashimiz mumkin. O'zgaruvchi saqlaydigan ma'lumot turi avtomatik tarzda aniqlanadi.

    typeof operatorlaridan foydalanib o'zgaruvchi saqlanayotgan ma'lumotning turini aniqlash imkoniyati mavjud.

    Object turdagi qiymat : O'zgaruvchi murojaat qilgan manzilda ko'rsatilgan boshqa manzilda saqlanadi
