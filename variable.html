"use strict" mode yordamida yozayotgan kodimizda xatolardan qochish imkoniyati mavjud

O'zgaruvchilarni e'lon qilish :
  • Javascript dasturlash tilda o'zgaruvchilarni 3 xil usulda elon qilish mavjud : let , const , var kalit so'zlaridan foydalanib amalga oshiriladi. let va const ES6'da taqdim etilgan kalit so'z hisoblanadi.

    • E'lon qilinib bo'lingan o'zgaruvchini qiymatini o'zgartirishda hech qanday kalit so'zdan foydalanilmaydi

    Let - dasturimiz ishlash davomida o'zgarish mumkin bo'lgan o'zgaruvchilarni e'lon qilish uchun foydalaniladi !

    Const - dasturimiz ishlashi davomida o'zgarmaydigan o'zgaruvchilarni e'lon qilish uchun foydalaniladi !
  • Const bu constanta ma'nosini bildiradi ya'ni o'zgarmas !
  • Const yordamida berilgan o'zgaruvchini qiymatini o'zgartirish mumkin emas agar o'zgartisak xatolik kelib chiqadi ya'ni "Type error" degan !

    • Var - ES6 ga qadar o'zgaruchilarni e'lon qilish uchun ishlatilgan kalit so'z "Let" bilan dayarli bir xil vazifani bajargan !
  • Let va Var o'xshash bo'lishiga qaramasdan ma'lum bir farqlari mavjud. Var yordamida e'lon qilingan o'zgaruvchining ko'rinish ko'lami (scope) funksiya darajasida bo'lsa , let esa blok darajasida korinadi

    • Java script yordamida hech qanday kelit so'zi ko'rsatilmasdan o'zgaruvchi yaratish imkoniyati mavjud lekin bu ko'rinishdagi o'zgaruvchilar yaratish tavsiya etilmaydi !
